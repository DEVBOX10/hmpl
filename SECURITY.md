# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >= 3.x   | yes |
| < 3.0.0   | no                |

## Reporting a Vulnerability

To report a vulnerability, please send an email with detailed information to hmpllang@gmail.com or open [issue](https://github.com/hmpl-language/hmpl/issues/new) on GitHub
